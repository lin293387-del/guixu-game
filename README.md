# 归墟游戏界面

这是一个用于酒馆助手的归墟游戏界面文件。

## 文件说明

- `guixu-game.html` - 主要的游戏界面文件
- `guixu-game-4.5.html` - 第4.5章版本的游戏界面

## 使用方法

在酒馆助手的正则替换中使用以下链接：

```html
<body>
  <script>
    $('body').load('https://cdn.jsdelivr.net/gh/lin293387-del/guixu-game@main/guixu-game.html')
  </script>
</body>
```

## CDN链接
